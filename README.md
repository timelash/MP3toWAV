Simple MP3 to WAV Converter

Instructions

	1.	Place the WAV file in the same directory as mp3towav.py.
	2.	Run the following command to convert your MP3 file to WAV format:

      python3 mp3towav.py {filename.mp3}

  Replace {filename.mp3} with the actual name of your MP3 file.

  Ensure you have Python 3 installed.
